HTML Input Types
Description Input example Notes
type="button" Needs value for button text
type="checkbox"
checkbox text only after input part
type="color" with value you can set default color. not supported in IE or safari 9.1 or earlier
type="date" IE and safari old versions not supported
type="email" IE9 and earlier not supported
type="file"
type="month" Firefox, safari and IE11 and earlier not supported
type="number" min and max attributes to add numeric restrictions in input field. not supported IE9 and earlier.
type="password" characters in the field are masked.
type="radio" Radiobutton text after input tag.
type="range" min and max can be used. IE9 and earlier not supported.
type="reset" if you change input value and then click reset button, the form data will be reset to default values.
type="search"
type="submit" needs e.g. PHP for sending data
type="tel"
type="text" most common input used
type="time" safari and IE12 or earlier not supported
type="url" IE9 and earlier not supported
type="week" firefox, safari or IE11 not supported
Select and text area
type="select" select is wrapping options
type="textarea" size of text area can be specified by the cols and rows attributes. or even better, through css height and width
